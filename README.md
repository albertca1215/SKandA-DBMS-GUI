Source code for SK&A Database, COSC 578, by Dylan Dulmaine, Zora Johnson, Samyog Panta, and Albert Cantoria.
TO-DO: Create executable .jar file, once everything is known to work.
Entry point file is SKA Database GUI\src\ska\SKADatabaseApp.java (Run this file on NetBeans).
REMINDER: In order to run this on NetBeans, you need the MySQL JDBC Driver. (In a NetBeans Project, go to Libraries (right click) -> Add Library -> "MySQL JDBC Driver"
Additionally, you have to run "sql\SKA Population Query.sql" in order to properly create and populate tables necessary for the GUI's functionality.
